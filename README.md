# BRO grondwater plugin
Een QGIS plugin om grondwaterstanden uit de BRO te downloaden.


LET OP: Dit package gaat numpy versie >=2.0 installeren. Dit conflicteert met QGIS 3.x versies.
Na installatie van de packages moet je daarom numpy weer verwijderen en terugzetten naar een lagere versie (zie issue #2)
Vanaf QGIS 4.x zou numpy 2.0 ondersteund moeten worden.

We wachten voor nu op die volgende versie en zetten dan de required versie naar 4.x
